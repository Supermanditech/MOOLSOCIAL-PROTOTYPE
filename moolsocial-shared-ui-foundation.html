Regression Rule Before production coding

Every New Screen Must Use

Brand tokens, logo lockup, 8px card radius, pill controls, decision-card data structure, always-on consumer access pattern and the approved Screen 04 Mool dock geometry.

  • Bottom dock must keep the same shape across screens: round Mool, center action rail, round Chat.
  • Do not create a new Mool icon, MS mark or alternate dock shape on later screens.
  • Middle dock labels must never be clipped. If there are four or fewer actions, fit them fully; use horizontal scroll only for larger action sets.

Do Not Add

Unapproved colours, decorative blobs, new brand marks, role-switch tabs, product grids as the main commerce experience, or vague marketplace copy.
