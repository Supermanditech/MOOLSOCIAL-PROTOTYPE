UX Verdict

Screen 08 is a public social composer first. It should feel familiar to users of X, Facebook, Threads, Instagram and YouTube, while staying MoolSocial-branded and cleaner than a creator studio.

Posting Model

MoolSocial should allow any signed-in user to post text, images, shorts and videos. Earning, campaigns, analytics and payouts require creator setup later, but posting itself should not be blocked.

Mobile + Web

Mobile supports camera, gallery and quick capture. Web later supports long-form upload, thumbnails, captions and creator studio, but the same Create entry remains the mental model.

Screen Contract What Screen 08 must preserve
Entry
User arrives from Shorts, Videos, Feed or future website create entry. No role switch is required.
Primary Modes
Text, Photo, Short, Video, Story, Live, Thread and Poll are the visible social creation modes for MVP.
Creator Logic
Everyone can post. Creator profile, monetisation, campaign access and payout setup are upgrade paths after account verification.
Business Logic
Product/service/work proof must not dominate this screen. It can appear later as an optional contextual attachment from verified workspace flows.
Dock
Mool, Shorts, Videos, Feed, Create and Chat remain one tap accessible. Create must never be clipped in the bottom dock.
Full Stack Contract API and backend logic for later production

Data Needed

  • user profile, handle, avatar, language, location permission and audience defaults.
  • post type, destination channel, text, media assets, captions, hashtags, mentions, schedule and draft state.
  • creator eligibility state: normal user, creator setup pending, verified creator, monetisation ready.

API Intent

  • POST /v1/social/drafts/upsert for autosave and manual draft save.
  • POST /v1/social/media/upload-url for photo, short and video upload.
  • POST /v1/social/create/submit for public post publish.
  • GET /v1/creator/status for creator earning and studio eligibility.

States To Design

  • text-only post, image selected, short selected, long video selected, story draft, live scheduled and thread draft.
  • uploading, paused upload, upload failed, unsafe content, copyright warning and offline draft.
  • creator setup available, creator setup pending and monetisation not yet eligible.
