# Brand Name Round 16 - New Criteria Reset

Date: 2026-06-25

Purpose:

Reset the SuperMandi app naming process after rejecting prior directions.

This round intentionally stops following the old criteria and naming habits.

## Old Criteria Retired

Do not continue with:

- Yonda / Omera direction
- random five-letter invented sounds
- names with no first-impression meaning
- Hindi-ending pattern names
- grocery / daily / mandi / delivery-first names
- family-letter juggling from Aarya and Vivaan
- role-specific names that feel only like taxi, grocery, creator, restaurant, marketplace, payments or social media
- names that only sound modern but do not carry emotional meaning

## New Core Question

The name should answer this feeling:

When any Indian user opens the app, whether consumer, auto captain, grocer, restaurant, manufacturer, creator, worker or professional, what should they feel?

Answer:

"Yahan se mera kaam seedha, asli, bharose ke saath ho jayega."

The name should not describe a marketplace.

The name should feel like a direct operating place where life, work, social, commerce, money and communication get done.

## New Naming Filters

### 1. Outcome Feeling First

The name must carry at least one of these emotions:

- kaam ho gaya / gets done
- asli / real / trusted
- seedha / direct / no confusion
- saath / people together
- barkat / growth / prosperity
- rahat / relief / ease

### 2. User-Neutral

The name must work equally for:

- individual consumer
- auto captain
- grocer
- restaurant
- manufacturer
- wholesaler
- creator
- home service worker
- professional
- finance or money user

If it sounds like only one category, reject it.

### 3. Direct Operating App, Not Listing App

Reject names that feel like:

- bazaar
- directory
- yellow pages
- ecommerce shelf
- classified ads
- passive listings

The name must feel active: ask, scan, pay, book, work, chat, post, fulfil.

### 4. Chaiwala Pronunciation Test

A small vendor should be able to say the name without embarrassment or correction.

Pass examples:

- two or three natural sounds
- open vowels
- no spelling puzzle
- no foreign tongue-twist

Reject names that need explanation before pronunciation.

### 5. Consumer Verb Test

The name should be able to enter daily speech:

- "___ pe kar de"
- "___ se mangwa"
- "___ pe daal"
- "___ se payment kar"
- "___ pe kaam aa gaya"

### 6. First-Impression Meaning Test

The user should sense something within 3 seconds:

- trust
- completion
- directness
- togetherness
- growth
- relief

If the name is emotionally blank, reject it.

### 7. Trademark Reality Test

Exact common words may feel good but are often crowded.

Before keeping any name in the real shortlist:

- IP India exact search
- IP India phonetic search
- MCA / company-name search
- Play Store search
- App Store search
- domain search
- social handle search
- class conflict check for 9, 35, 36, 38, 39, 41, 42, 43 and 45

## New Direction

The next names should come from "meaningful outcome roots", not random sounds.

Priority roots:

- ho jaye / done
- asli / real
- seedha / direct
- poora / complete
- pakka / sure
- hazir / available
- zariya / the way
- raunak / lively growth
- barkat / prosperity
- rahat / relief

These roots are not final names. They are emotional territories.

The next round should generate names from these territories but avoid exact crowded words where possible.
