# Brand Name Round 8 - Sound Logo / Icon Logo

User direction:

Keep hunting with completely new parameters and avoid earlier parameters.

Parked fallback names:

- Yonda
- Omera

This round does not optimize for:

- Hindi-root meaning
- grocery/daily meaning
- trust/common-word meaning
- action/resolve meaning
- social-only meaning
- marketplace/local-place meaning
- neutral platform naming
- warm emotional naming

## New Parameters

This round uses a sound-logo and icon-logo test.

The name should:

- look strong as a Play Store icon
- work as a one-word sticker on autos, shop counters, delivery bags and QR stands
- work as a creator watermark
- work on a manufacturer/business dashboard
- be short enough for mass recall
- have a distinctive visual letter shape
- not need explanation on day one
- be extendable into product surfaces later

Example extension test:

- Name Pay
- Name Work
- Name Business
- Name Captain
- Name Creator
- Name Social
- Name Chat

## Current Strongest Recommendation

**Veyu**

Reason:

It has the cleanest mass-mouth balance in this round. It is short, not category-specific, not overly soft, and can sit across consumer, creator, captain, manufacturer, business, money and work surfaces.

It feels less emotionally empty than Yonda because the V/Y sound gives movement and modernity, but it is still not trapped by a literal meaning.
