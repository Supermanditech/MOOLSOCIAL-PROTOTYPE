# Brand Name Round 27 - Memory-First Invented Names

Working note only. Not final brand clearance.

## Reset

Previous rounds were still too close to visible naming patterns:
- Hindi-ending names
- soft premium names
- generic Indian words
- platform/descriptive names
- obvious socio-commerce abstractions
- names that sound like naming-consultant outputs

This round uses a different test:

> Can an ordinary Indian user remember it after hearing it once, say it without spelling help, and use it naturally as a daily action?

Street tests:
- "___ pe dekh le."
- "___ kar de."
- "___ se payment bhej."
- "___ pe kaam mil gaya."
- "Mummy bhi ___ use kar legi."
- "Captain ko ___ se ride mili."
- "Manufacturer ko ___ se order mila."

The name does not need to explain the business model. It needs to become the action.

## New Naming Principles

1. 2 syllables preferred.
2. 4-5 letters preferred.
3. Avoid obvious category words.
4. Avoid clever abbreviations.
5. Avoid too much meaning.
6. Use Indian-mouth phonetics, not only Hindi words.
7. Must be easy for consumer, captain, worker, retailer, creator and manufacturer.
8. Must feel usable for money, work, commerce, social and chat.
9. Must be brandable visually as a premium mass app icon.
10. Legal search comes after emotional shortlist.

## Candidate Names

### A. Outcome / "It Gets Done" Territory

1. **Hoja**
   - Memory hook: "ho jaayega" without writing the full phrase.
   - Usage: "Hoja pe dekh le", "Hoja se payment bhej", "Hoja pe kaam mil gaya."
   - Strength: extremely Indian-mouth, instant recall, outcome-led.
   - Risk: informal; needs premium visual treatment to avoid casualness.

2. **Hova**
   - Memory hook: outcome + value, but more invented than Hoja.
   - Usage: "Hova kar de", "Hova pe order daal."
   - Strength: compact, app-like, less literal.
   - Risk: meaning is empty unless brand builds it.

3. **Hozo**
   - Memory hook: "ho" outcome with modern sound.
   - Usage: "Hozo pe dekh", "Hozo se kaam mila."
   - Strength: distinctive, logo-friendly, startup-scale.
   - Risk: slightly more foreign than Hoja.

4. **Hodo**
   - Memory hook: "ho" + "do", outcome plus action.
   - Usage: "Hodo kar de."
   - Strength: very verbable.
   - Risk: could sound too playful if not designed carefully.

### B. Take / Get / Bring Territory

5. **Leko**
   - Memory hook: sounds like "le ko", meaning take/get without being exact Hindi.
   - Usage: "Leko pe dekh le", "Leko se manga le."
   - Strength: strong public recall, useful across commerce and money.
   - Risk: slightly mass-market; premium system design must carry it.

6. **Lavo**
   - Memory hook: bring/get movement.
   - Usage: "Lavo se bhej", "Lavo pe order."
   - Strength: delivery, money, work and commerce compatible.
   - Risk: may lean logistics if over-explained.

7. **Pavo**
   - Memory hook: pao/get, but softer and more brandable.
   - Usage: "Pavo pe mil jayega."
   - Strength: simple, national-mouth sound.
   - Risk: needs search because it may appear in food/finance contexts.

8. **Molo**
   - Memory hook: "mol" means value/price in many Indian contexts.
   - Usage: "Molo pe rate dekh", "Molo se payment bhej."
   - Strength: commerce and value-chain fit.
   - Risk: price/value association may dominate.

### C. Public Call / Everyday Signal Territory

9. **Ajo**
   - Memory hook: short public call, "aa jo" energy without exact phrase.
   - Usage: "Ajo pe aa ja", "Ajo pe dekh."
   - Strength: highly memorable, social-friendly.
   - Risk: can feel too casual unless visual identity is premium.

10. **Talo**
    - Memory hook: rhythm, taal, system, coordination.
    - Usage: "Talo pe kaam mila."
    - Strength: premium mass, easy pronunciation.
    - Risk: meaning is subtle.

11. **Ravo**
    - Memory hook: public voice, response, movement.
    - Usage: "Ravo pe bol", "Ravo pe dekh."
    - Strength: strong for social plus commerce.
    - Risk: needs trademark search; may have global use.

12. **Sabo**
    - Memory hook: "sab" but not generic "sabko/sabka".
    - Usage: "Sabo pe sab milta hai."
    - Strength: inclusive, mass India, super-app compatible.
    - Risk: close to common word and surname/business use.

### D. Premium Mass Coined Territory

13. **Kivo**
    - Memory hook: quick, app-like, strong consonants.
    - Usage: "Kivo kar de", "Kivo pe dekh."
    - Strength: brandable, modern, easy.
    - Risk: emotionally emptier until brand meaning is built.

14. **Tavo**
    - Memory hook: Indian-mouth, short, non-category.
    - Usage: "Tavo pe order daal", "Tavo se paise bhej."
    - Strength: flexible masterbrand.
    - Risk: less immediate emotional meaning.

15. **Kavo**
    - Memory hook: kaam/value/open commerce feel.
    - Usage: "Kavo pe kaam mil gaya."
    - Strength: work plus commerce compatible.
    - Risk: may feel like a generic startup name if visual identity is weak.

16. **Rimo**
    - Memory hook: rhythm + motion.
    - Usage: "Rimo pe dekh", "Rimo se ride mili."
    - Strength: soft but not weak.
    - Risk: could be confused with remote/remit if positioned badly.

17. **Tiko**
    - Memory hook: tick/done, quick action.
    - Usage: "Tiko kar de."
    - Strength: very sticky, completion signal.
    - Risk: may feel too small unless brand system is serious.

18. **Miko**
    - Memory hook: friendly, memorable, creator/social compatible.
    - Usage: "Miko pe post kar", "Miko pe order dekh."
    - Strength: warm mass app sound.
    - Risk: possibly too soft/cute.

### E. Indian-Scale But Not Literal Territory

19. **Amro**
    - Memory hook: Indian sound, broad and stable.
    - Usage: "Amro pe dekh."
    - Strength: serious, national brand potential.
    - Risk: could feel too close to existing personal/company names.

20. **Nelo**
    - Memory hook: easy, clean, near/network feel.
    - Usage: "Nelo pe mil jayega."
    - Strength: simple and premium.
    - Risk: may feel global-empty without story.

21. **Daro**
    - Memory hook: strong two-syllable public recall.
    - Usage: "Daro pe dekh."
    - Strength: bold logo sound.
    - Risk: "dar" means fear in Hindi; probably not ideal.

22. **Javo**
    - Memory hook: movement, jao/go, but invented.
    - Usage: "Javo se ride mili", "Javo pe kaam."
    - Strength: mobility plus work fit.
    - Risk: may lean travel/ride too much.

23. **Voko**
    - Memory hook: voice/work/commerce sound.
    - Usage: "Voko pe bol", "Voko pe order."
    - Strength: social and operating layer compatible.
    - Risk: more foreign.

24. **Beko**
    - Memory hook: bold, simple, one-hearing recall.
    - Usage: "Beko pe dekh."
    - Strength: easy for all users.
    - Risk: likely existing global appliance brand conflict.

## Current Best Fresh Directions

### 1. Hoja
This is the strongest mass-memory candidate in this round.

Why:
- It carries the core promise: things happen.
- It can become a verb naturally.
- It fits consumer, work, money, ride, commerce, social and fulfilment.
- It feels Indian without locking to religion, caste, region or category.

Concern:
- It is informal. The logo, typography and product design must make it feel premium.

### 2. Leko
Strong second direction.

Why:
- It feels like get/take/order/pay without becoming a direct word.
- It can travel across commerce, money and services.
- It is very easy to speak.

Concern:
- It may feel too transactional unless social/work layers are designed strongly.

### 3. Tavo
Strong masterbrand direction.

Why:
- It is clean, invented and flexible.
- It does not trap the company in grocery, taxi, social or fintech.
- It can look premium.

Concern:
- It needs brand meaning to be built from scratch.

### 4. Sabo
Strong inclusive direction.

Why:
- It hints "for everyone" without saying Sabka.
- It is easy for the public.
- It supports the super-app idea.

Concern:
- It may face common-word/surname/business conflicts.

### 5. Hozo
Modern alternative to Hoja.

Why:
- It keeps outcome energy but becomes more ownable.
- It is more startup/global than Hoja.

Concern:
- Less emotionally immediate for Indian mass users.

## Recommendation From This Round

Do not pick from all 24.

Only test these 5 aloud:

1. **Hoja**
2. **Leko**
3. **Tavo**
4. **Sabo**
5. **Hozo**

Best instinct candidate: **Hoja**

Best premium-masterbrand candidate: **Tavo**

Best commerce-action candidate: **Leko**

Best inclusive-superapp candidate: **Sabo**

Best modern-ownable candidate: **Hozo**
