# Screen 08: Social > Create - Approved

Approval date: 2026-07-07

Approved direction:
- Screen 08 is a pure public social creation hub, not a proof/business operations screen.
- Any signed-in user can post text, photos, shorts, videos, stories, live sessions, threads and polls.
- Creator earning is an upgrade path, not a posting barrier.
- The visible creator path must clearly support Studio, monetisation, campaigns, analytics and payouts after verification.
- Create must support both mobile app creation and future website/web creator upload flow.
- Product/service/work proof can exist later only as optional contextual attachment from verified workspace flows; it must not dominate this screen.

Approved visible modes:
- Text
- Photo
- Short
- Video
- Story
- Live
- Thread
- Poll

Approved creator toolkit:
- Templates
- Captions
- Cover
- Collab
- Live
- Insights

Regression lock:
- Bottom dock must keep Mool, Shorts, Videos, Feed, Create and Chat visible.
- Create tab must never be clipped.
- Screens 05, 06 and 07 must link to Screen 08 from their Create action inside the approved-final baseline.
