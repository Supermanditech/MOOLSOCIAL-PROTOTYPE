# Brand Name Round 33 - Availability Scan For Round 32 Names

Working note only. Not legal clearance.

Date: 2026-06-25

## Scope

Checked the 28 names from Round 32 using public web search, visible trademark aggregator pages, company-directory/MCA mirror pages, app/website/social-handle signals, and known public directories.

Important limitation:
- Official IP India Public Search requires CAPTCHA/manual search.
- Official MCA company-name search also requires live portal/manual search and subjective similarity review.
- Therefore this note classifies names as:
  - **Reject**: visible direct conflict or very high risk.
  - **Avoid**: not clean enough for a masterbrand.
  - **Hold**: no major visible conflict in this scan, but official IP/MCA exact + phonetic search required.
  - **Cleaner**: comparatively fewer visible conflicts, still not legally cleared.

Official final checks required:
- IP India Public Search: https://tmrsearch.ipindia.gov.in/tmrpublicsearch/
- MCA name check: https://www.mca.gov.in/
