# Brand Name Shortlist - 20 Candidate Names

Status: working note only. Not final legal advice. Do not update final screenbook visuals until name is approved.

## Naming Objective

Create a trademarkable, mass-recall Indian super-app brand under `SuperMandi Tech Private Limited`.

The name must work for:

- Consumer
- Household
- Auto captain
- Grocer
- Restaurant
- Doctor/clinic
- Manufacturer
- Worker
- Creator
- Offline partner

The name should become spoken like:

- "___ kar de"
- "___ pe daal de"
- "___ se book kar"
- "___ se kaam ho jayega"

## Research Notes

Preliminary web search only. Official trademark clearance still required through IP India wordmark + phonetic search across relevant classes.

- `Jodo` has an existing fintech/app presence.
- `Aayo` appears as an app handle.
- `Rozzo` appears as an app in Nashik.
- `Nipto` is already an app name.

- `Hojay` has an Indian LLP/company result, so it needs extra checking.

## Top Recommendation

### 1. Pakko

Meaning: confirmed, reliable, sure.

Why it fits:

- Consumer: "Pakko se book kar."
- Auto captain: "Pakko ride."
- Grocer: "Pakko order."
- Manufacturer: "Pakko buyer demand."
- Strong trust/verification fit.

Risk:

- Derived from common "pakka", so official phonetic search is necessary.

Verdict:

Best emotional fit for SuperMandi's proof, trust, readiness and outcome model.

## My Current Pick

`Pakko`

Reason:

SuperMandi's real difference is not only speed. It is verified, funded, proof-based, payment-safe, fulfilment-ready outcome. `Pakko` communicates certainty to a consumer, driver, retailer, manufacturer and worker without sounding only like grocery, only like jobs or only like delivery.

Possible public line:

`Pakko - Pay. Buy. Book. Work.`

Possible promise line:

`Kaam, order aur payment - Pakko.`
