# Brand Name Round 17 - New Criteria Candidates

Date: 2026-06-25

Purpose:

Fresh name round after retiring old criteria.

This round prioritizes first-impression meaning over empty invented sound.

Core feeling:

"Yahan mera kaam seedha, asli, bharose ke saath ho jayega."

## Stronger Direction

The strongest territory is not "market", "delivery", "daily", "social" or "business".

The strongest territory is:

**Kaam ho gaya.**

For India, "kaam" is broader than work. It can mean payment, booking, order, ride, repair, post, chat, proof, service, delivery, deal, creator task, retailer sale, manufacturer fulfilment, or family need.

## Current Top Candidates

| Name | First Impression | Why It Fits | Main Risk |
|---|---|---|---|
| Seedho | Direct / straight / no middleman | Strong for direct operating app. Feels simple, clean, one-tap, no confusion. | Needs official IP India phonetic search against Sidho/Seedha/Seekho style marks. |
| KaamHo | Work/need gets done | Most literal match to the SuperMandi outcome. Easy to explain: "Har kaam ho." | Looks more phrase-like than brand-like; needs logo treatment to feel premium. |
| HoGaya | Done / completed | Very strong mouth recall. Anyone understands it instantly. | Generic phrase; may be harder to own strongly as a word mark. |
| HoJaye | Let it happen / get it done | Softer than HoGaya; good for ask/voice/AI/action flow. | Common advertising phrase; may be weak legally. |
| Kamo | Compressed from Kaam Ho | Short, modern, 4 letters, easy app icon name. | Meaning is not instant unless brand story/logo teaches it. |
| SahiHo | Done right / correct | Trust + outcome in one phrase. | Sahi is crowded/common; must be checked carefully. |
| PakkaHo | Confirmed / sure / done | Good for trust, payment, fulfilment, bookings. | Slightly long and phrase-like. |
| PooraHo | Complete / fulfilled | Strong for end-to-end operating system. | Slightly formal; may not sound like a daily verb. |
| KaamSet | Work is set | Indian speech friendly; useful for merchant/work/money. | Could feel work-only, less social/emotional. |
| SabSet | Everything set | Strong everyday phrase; broad across all users. | Can visually read like "subset"; "set hai" related marks/usages need check. |
