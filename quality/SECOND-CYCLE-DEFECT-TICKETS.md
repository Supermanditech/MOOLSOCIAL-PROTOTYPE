# Second-Cycle Micro-Interaction Defect Tickets

## Audit rule

These tickets belong to the independent second audit cycle started from the deployed `8578b99` build. A ticket is closed only after the original failed interaction is replayed from a clean state, the complete affected journey passes, and the final full regression passes at mobile and laptop viewports.
